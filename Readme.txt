Dibangun dengan Tailwind CSS untuk gaya tampilan yang responsif dan elegan, serta dilengkapi dengan komponen Preline UI untuk mempercepat pengembangan antarmuka. Proyek ini juga memanfaatkan ikon-ikon dari SVGRepo dan Feather Icons untuk menciptakan tampilan yang menarik dan intuitif. Selain itu, kami menggunakan foto-foto dari Unsplash yang bebas lisensi, menjamin penggunaan visual yang aman dan berkualitas tinggi dalam proyek ini.
